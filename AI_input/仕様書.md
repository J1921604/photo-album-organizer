# 仕様

写真を別々のアルバムに整理できるアプリケーションを作成する。
アルバムは日付ごとにグループ化され、メインページ上で日付または名前ソート操作により再編成できる。
アルバムは他のアルバムにネストされない。
各アルバム内では、写真がタイル状のインターフェースでプレビュー表示される。
サムネイルをクリックするとフルサイズ表示され、フルサイズでダウンロードできる。

# 計画

アプリケーションは最小限のライブラリでViteを使用する。
標準のHTML、CSS、JavaScriptを活用する。
画像はどこにもアップロードされず、メタデータはローカルのSQLiteデータベースに保存する。
